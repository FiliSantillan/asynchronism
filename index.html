Rick & Morty API

JS es un lenguaje de programación asíncrono y no bloqueante con un manejador de eventos mejor conocido como el Event Loop, implementado en un único hilo para sus interfaces de entrada y salida. El asincronismo en simples palabras es la acción que no ocurre al mismo tiempo.

En este pequeño ejercicio se consume la API de Rick & Morty, de tres distintas maneras: Callbacks, Promises y Async / Await.
